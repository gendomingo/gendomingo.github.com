Inheritance in CSS

CSS Inheritance is a very useful approach to CSS language as it greatly reduces time and effort to assign different properties or style to each element or what they call as the “child elements.” Instead of duplicating CSS rules to each child elements, then it would be best to assign the properties to the parent elements with inherit if they all have the same properties. An example of CSS Inheritance are color, font-family, line-height.

Specificity in CSS

Specificity in CSS helps defines which property of an element to use if two or more properties are applied to an element. When specificity determines whether which one is more or less important when there are conflicting rules, it overrides it. A compound selector is used to specify a specific element to style while multiple selector is used to a group of elements to style.

ID and Class Attributes

Both class and id are two selectors used in CSS that can be used to apply for styling to the HTML elements in a web page. The main difference is class selector can be used to apply to a group of elements while id selector is used to apply to one single unique element.
